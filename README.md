# !!TekagiUI

## Overview
!!TekagiUI is a lightweight World of Warcraft UI addon that provides user interface enhancements.


## Feats
This addon is designed to improve the WoW interface by giving you:
- Automatic pixel perfect scaling.
- Class/reaction colours for default Player, Target, Focus, Target of Target frames.
- Enable shorthand commands for:
    - /rl to reload
    - /lg to leave group
    - /rc to ready check

## Notes
Place this folder in your World of Warcraft `_retail_` Interface/AddOns directory and enable it from the in-game AddOns menu.
